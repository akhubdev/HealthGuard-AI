# HealthGuard AI

HealthGuard AI is an AI-powered healthcare analytics system that predicts diabetes risk using machine learning.

## Features

- Diabetes risk prediction using machine learning
- Admin dashboard with analytics
- Prediction history tracking
- PDF report generation
- Cloud deployment

## Tech Stack

- Python
- Flask
- Scikit-learn
- SQLite
- HTML
- CSS
- Render Cloud
